一、引用类型

1、indexOf()，全等运算符（===）

因为 indexOf() 会执行 === 操作，全等操作符会比较两个值的 地址 ，所以以上返回 -1。

2、值传递，址传递（这一块可能还是理解的有问题，暂时这么理解吧，后面再来修改）

函数参数的传递都是值传递，引用类型的赋值都是址传递。sum = null 是sum被重新赋了新的地址，不是修改了原方法，所以 anotherSum()不受影响，还是指向之前的内存地址。
